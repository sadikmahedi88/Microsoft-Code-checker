📌 About The Project

Microsoft Code Checker is a powerful and multi-threaded tool designed to validate Microsoft redeem codes efficiently.

It supports multiple accounts, proxies, and provides categorized results such as:

- ✅ Valid codes
- 💳 Valid (requires payment method)
- 🌍 Region locked
- ❌ Invalid / Expired / Redeemed

---

⚙️ Features

- ⚡ High-speed multi-threading
- 🔐 Microsoft account login support
- 🌐 Proxy support (optional)
- 📂 Auto-sorted results
- 🧠 Smart error & rate-limit handling
- 📊 Live statistics in console

---

🛠️ Installation

git clone https://github.com/sadikmahedi88/Microsoft-Code-checker.git
cd Microsoft-Code-checker
pip install -r requirements.txt

---

▶️ Usage

python main.py

---

📂 Input Files

Make sure to prepare these files:

"accounts.txt"

email1:password1
email2:password2

"codes.txt"

XXXXX-XXXXX-XXXXX
XXXXX-XXXXX-XXXXX

"proxies.txt" (optional)

ip:port
ip:port:user:pass
user:pass@ip:port

---

📁 Output

Results will be saved automatically inside:

results/
└── check_TIMESTAMP/
    ├── valid_codes.txt
    ├── valid_cardrequired_codes.txt
    ├── invalid.txt
    ├── region_locked_codes.txt
    ├── unknown_codes.txt
    └── summary.txt
